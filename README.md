# Object-Oriented Programming with Python (BT151CO)

Course repository for **BT151CO — Object-Oriented Programming**, B.Tech. in AI, Purbhanchal  University .

This repository contains:

* Lecture slides (PDF)
* Jupyter Notebook examples (`.ipynb`)
* Assignments
* Assignment solutions (when released)
<!-- * Additional reference materials -->

---

## Course Objectives

Upon successful completion of this course, students will be able to:

* Apply object-oriented programming principles and practices using Python.
* Design and develop graphical user interfaces (GUIs) using Tkinter.
* Build database-driven applications using Python and SQL.
* Handle exceptions, files, collections, networking, regular expressions, and multithreading in Python.
