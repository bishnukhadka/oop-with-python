# Object-Oriented Programming ‚Äî BT151CO

## Course Objectives

This course is designed to equip students with the following skills:
- Object-oriented programming (OOP) principles and practices in Python
- Designing Graphical User Interfaces (GUIs) using Python
- Building database-driven applications with Python
